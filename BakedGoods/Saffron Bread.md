# Bread Machine Saffron Bread

[From Here](https://nowyourecookingrecipes.com/bread-machine-saffron-bread/)

> A bread machine version of a historic recipe from England. It is yellow, has raisins, and is delicious toasted with butter and honey.

## Ingredients

| Ingredient          | Qty    | Unit  | Notes                                                |
|:--------------------|:------:|:-----:|:-----------------------------------------------------|
| Milk                | 1      | cup   |                                                      |
| Ground Saffron      | 1/8    | tsp   | (1 package)                                          |
| Butter              | 1      | Tbsp  | Softened                                             |
| Eggs                | 2      |       |                                                      |
| Sugar               | 1/3    | cup   |                                                      |
| All-Purpose Flour   | 3 1/4  | cups  |                                                      |
| Salt                | 1      | tsp   |                                                      |
| Yeast               | 1      | pkg   | Bread maker yeast                                    |
||||
| Raisins             | 3/4    | cup   |                                                      |

## Instructions

1.  **Load Machine:** Add all ingredients, except raisins, to the bread maker pan in the order listed (or as directed in your bread maker instructions).
2.  **Start:** Set bread maker to the regular setting and start.
3.  **Add Raisins:** Add the raisins when the beeper sounds to add additional ingredients.

**Notes:**

* **Size:** This makes about a 1 1/2 pound loaf.
* **Flour:** The recipe calls for standard flour (AP), but bread flour can also be used.
* **Serving:** Great toasted with butter and honey.
