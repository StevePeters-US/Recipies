# Waffle Mix

## Dry Ingredients
| Ingredient          | Qty   | Unit  | Notes               |
|:--------------------|:-----:|:-----:|:--------------------|
| All-purpose flour (AP) | 1   | cup   |                    |
| Whole wheat flour   | 1    | cup   |                    |
| Baking soda         | 1/2  | tsp   |                    |
| Baking powder       | 1    | tsp   |                    |
| Salt                | 1    | tsp   |                    |
| Sugar               | 3    | tbsp  |                    |

## Wet Ingredients
| Ingredient          | Qty    | Unit  | Notes               |
|:--------------------|:------:|:-----:|:--------------------|
| Eggs                | 3      |       |                    |
| Butter (unsalted)   | 2      | oz    |                    |
| Buttermilk          | 16     | oz    |                    |

## Instructions
Mix wet and dry ingredients seperately. Combine - will be lumpy. Overmixing will make waffles tough.
