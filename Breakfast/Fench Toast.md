# French Toast

## Ingredients

| Ingredient          | Qty    | Unit  | Notes                                                                                 |
|---------------------|--------|-------|---------------------------------------------------------------------------------------|
| Egg                 | 5      |       |                                                                                       |
| Half and Half       | 1/2    | cup   |                                                                                       |
| Honey               | 2      | Tbsp  |                                                                                       |
| Salt                | 1/4    | tsp   |                                                                                       |

## Instructions

Throw in a crock pot for 6 - 8 hours.
