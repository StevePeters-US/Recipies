# Quiche

## Ingredients

| Ingredient          | Qty    | Unit  | Notes                                                                                 |
|---------------------|--------|-------|---------------------------------------------------------------------------------------|
| Pie Crust           | 1      |       |                                                                                       |
| Egg                 | 4      |       |                                                                                       |
| Half and Half       | 1      | cup   |                                                                                       |
| Salt and Pepper     |        |       |                                                                                       |


## Add ins

| Ingredient          | Qty    | Unit  | Notes                                                                                 |
|---------------------|--------|-------|---------------------------------------------------------------------------------------|
| Onions              |        |       | sauteed                                                                               |
| Mushrooms           |        |       |                                                                                       |
| Spinach             |        |       |                                                                                       |
| Bacon               |        |       |                                                                                       |
| Cheese              |        |       | greyer, cheddar, or feta                                                              |

## Instructions

Pre-bake crust according to directions. Add scrambled eggs, half and half, and any add ins and bake at 375f for 45 minutes.
