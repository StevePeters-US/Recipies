# Garlic Aioli

## Ingredients

| Ingredient          | Qty    | Unit  | Notes                                                                                 |
|---------------------|--------|-------|---------------------------------------------------------------------------------------|
| Mayonnaise          | 3/4    | cup   |                                                                                       |
| Garlic              | 3      | clove |                                                                                       |
| Lemon Juice         | 2 1/2  | Tbsp  |                                                                                       |
| Salt                | 3/4    | tsp   |                                                                                       |
| Pepper              | 1/2    | tsp   |                                                                                       |

## Instructions

Use flat of knife to grind garlic and salt together. Mix ingredients together.
