# Acorn Squash

## Ingredients

| Ingredient          | Qty    | Unit | Notes                                                                                 |
|---------------------|--------|------|---------------------------------------------------------------------------------------|
| Acorn Squash        | 1      |      |                                                                                       |
| Butter              | 2      | Tbsp |                                                                                       |
| Maple Syrup         | 1      | Tbsp |                                                                                       |
| Salt                |        |      |                                                                                       |
| Pepper              |        |      |                                                                                       |
| Nutmeg              |        |      |                                                                                       |

## Instructions

Cut in half. Deseed. Roast on a greased cookie sheet at 350f for 25 minutes. Flip, fill with butter and seasonings. Roast an additional 10 minutes.
