# Chicken and Rice

| Ingredient          | Qty    | Unit  | Notes                                                                                 |
|---------------------|--------|-------|---------------------------------------------------------------------------------------|
| Brown Rice          | 2      | cup   |                                                                                       |
| Chicken (cooked)    | 2      | cup   | Chopped                                                                               |
| Carrot              | 2      |       | Sliced                                                                                |
| Shallot             | 1      |       | Minced                                                                                |
| Celery              | 1      |       | Sliced                                                                                |
| Parsley             | 1/2    | cup   |                                                                                       |
| Chicken Stock       | 2      | cup   |                                                                                       |
| Corn                | 1      | can   |                                                                                       |
| CreamofChicken Soup | 1      | can   | 10.5 oz can                                                                           |
| Salt                |        |       |                                                                                       |
| Tyme                |        |       |                                                                                       |
| White Pepper        |        |       |                                                                                       |

## Instructions

Cook rice as directed. Chop leftover chicken, combine with other ingredients and simmer for 15 minutes.
