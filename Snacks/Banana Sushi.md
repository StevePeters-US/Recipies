# Banana Sushi

## Ingredients
| Ingredient      | Qty  | Unit  | Notes  |
|:----------------|:----:|:-----:|:-------|
| Banana          | 1    |       | Sliced  |
| Wheat tortilla  | 1    |       |        |
| Almond butter   |      |       |        |


## Instruction
Spread almond butter on the tortilla and roll up with banana slices. Slice this into 1" disks
